# Hello World Tool

## 描述
Hello World 示例工具，演示 Agent 工具的基本结构和使用方法。

## 功能
- 接收 `name` 参数，返回问候消息
- 演示 ToolResult 的正确使用方式
- 零依赖，开箱即用

## 使用示例

```python
# 在 Agent 中调用
tool = "hello-world"
args = {"name": "Agent"}
# 输出: Hello, Agent! Welcome to Agent Toolkit!
```

## 安装
通过 agent_toolkit App 一键安装，或手动下载 `hello_world.py` 到本地。

## 依赖
无
